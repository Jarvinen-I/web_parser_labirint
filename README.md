### web_parser_labirint

Программа получает с сайта интернет-магазина книг следующие данные: 
- название книги;  
- автор;  
- издательство;  
- цена со скидкой;  
- цена без скидки;
- размер скидки;  
- наличие на складе.

Далее программа записывает полученные данные в .csv и .json-файлы с текущей датой.
